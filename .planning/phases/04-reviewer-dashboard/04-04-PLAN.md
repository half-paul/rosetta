<objective>
Build the review pane — the split-pane view where reviewers examine individual claims, verify sources, edit commentary, flag new content, and take approve/reject actions.

    Import db, schema tables (claims, commentaries, paragraphs, sections, articles, auditLog, users), eq/desc from drizzle-orm.

    Export `getCommentaryWithClaim(claimId: string)`:
    - SELECT: claim fields (id, text, severity, confidenceScore, charOffsetStart, charOffsetEnd, explanation, paragraphId), commentary fields (id as commentaryId, draftText, status, suggestedSources, assignedTo, createdAt), paragraph.content, section.title as sectionTitle, section.articleId, article.title as articleTitle
    - JOIN: claims -> commentaries (eq commentaries.claimId, claims.id) -> paragraphs -> sections -> articles
    - WHERE: eq(claims.id, claimId)
    - Return first row or null

    Export `getAuditFeed(claimId: string)`:
    - SELECT: auditLog fields (id, action, beforeState, afterState, createdAt), users.name as reviewerName, users.image as reviewerImage
    - LEFT JOIN: auditLog -> users on auditLog.reviewerId = users.id
    - WHERE: eq(auditLog.claimId, claimId)
    - ORDER BY: auditLog.createdAt DESC
    - Return array

    **2. Create src/app/dashboard/review/[claimId]/page.tsx** — Server Component:
    ```typescript
    import { auth } from '@/auth'
    import { redirect } from 'next/navigation'
    import { getCommentaryWithClaim, getAuditFeed } from '@/features/reviews'
    import { SplitPane } from '@/components/dashboard/split-pane'

    export default async function ReviewPage({
      params,
    }: {
      params: Promise<{ claimId: string }>
    }) {
      const session = await auth()
      if (!session?.user) redirect('/login')

      const { claimId } = await params  // MUST await — Next.js 16
      const data = await getCommentaryWithClaim(claimId)
      if (!data) redirect('/dashboard')

      const auditFeed = await getAuditFeed(claimId)

      return (
        <SplitPane
          data={data}
          auditFeed={auditFeed}
          reviewerId={session.user.id!}
        />
      )
    }
    ```
